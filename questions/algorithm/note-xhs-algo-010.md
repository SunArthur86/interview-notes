# 【拼多多 Java服务端】力扣198：打家劫舍

> 来源：拼多多复活赛一面面经（小红书）

## 一、费曼类比

想象你是一个**小偷站在一排房屋前**：

```
房屋编号:  0    1    2    3
金额:     [2,   7,   9,   3,   1]
           ↑              ↑
         偷这个        也偷这个 → 2+9+1=12
              ↑         ↑
           偷这个    也偷这个 → 7+3=10
```

规则：**相邻房屋不能同时偷**。你要决定站在每家门口"偷还是不偷"。

## 二、第一性原理分析

**核心问题**：在"不能选相邻元素"的约束下，求子序列的最大权和。

对第i家房屋，只有两个选择：

```
选择A: 偷第i家 → 第i-1家不能偷 → 收益 = dp[i-2] + nums[i]
选择B: 不偷第i家 → 收益 = dp[i-1]（继承到上一家为止的最佳）

最优决策: dp[i] = max(选择A, 选择B)
```

## 三、详细答案

### 解法一：标准DP（O(n)空间）

```java
class Solution {
    public int rob(int[] nums) {
        if (nums == null || nums.length == 0) return 0;
        if (nums.length == 1) return nums[0];
        
        int n = nums.length;
        int[] dp = new int[n];
        dp[0] = nums[0];
        dp[1] = Math.max(nums[0], nums[1]);
        
        for (int i = 2; i < n; i++) {
            dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
        }
        
        return dp[n - 1];
    }
}
```

### 解法二：空间优化（O(1)空间，推荐）

```java
class Solution {
    public int rob(int[] nums) {
        if (nums == null || nums.length == 0) return 0;
        if (nums.length == 1) return nums[0];
        
        int prev2 = nums[0];                    // dp[i-2]
        int prev1 = Math.max(nums[0], nums[1]); // dp[i-1]
        
        for (int i = 2; i < nums.length; i++) {
            int curr = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = curr;
        }
        
        return prev1;
    }
}
```

## 四、例子演示

```
输入: nums = [2, 7, 9, 3, 1]

逐步推演:
i=0: dp[0] = 2
i=1: dp[1] = max(2, 7) = 7
i=2: dp[2] = max(dp[1], dp[0]+9) = max(7, 2+9) = max(7, 11) = 11
i=3: dp[3] = max(dp[2], dp[1]+3) = max(11, 7+3) = max(11, 10) = 11
i=4: dp[4] = max(dp[3], dp[2]+1) = max(11, 11+1) = max(11, 12) = 12

输出: 12 （偷第0, 2, 4号房屋: 2+9+1=12）

状态表:
┌─────┬──────┬────────────────────────────┐
│  i  │nums[i]│         dp[i]              │
├─────┼──────┼────────────────────────────┤
│  0  │  2   │  2                         │
│  1  │  7   │  max(2, 7) = 7             │
│  2  │  9   │  max(7, 2+9) = 11          │
│  3  │  3   │  max(11, 7+3) = 11         │
│  4  │  1   │  max(11, 11+1) = 12        │
└─────┴──────┴────────────────────────────┘
```

## 五、扩展知识

### 环形打家劫舍（力扣213）

```
环形: 第一家和最后一家也相邻
策略: 拆成两个线性问题
  - 问题1: 偷 [0, n-2]（不偷最后一家）
  - 问题2: 偷 [1, n-1]（不偷第一家）
  - 取两者较大值
```

### 树形打家劫舍（力扣337）

```java
// 返回 int[2]: [不偷当前节点的最大收益, 偷当前节点的最大收益]
int[] dfs(TreeNode root) {
    if (root == null) return new int[]{0, 0};
    int[] left = dfs(root.left);
    int[] right = dfs(root.right);
    int notRob = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);
    int rob = root.val + left[0] + right[0];
    return new int[]{notRob, rob};
}
```

## 六、苏格拉底式面试提问

1. **"为什么贪心算法（每步选最大的）不能解决？"** — 因为局部最优不等于全局最优，如 [2,1,1,2] 贪心选2后错过末尾的2
2. **"你的空间优化为什么只需要两个变量？"** — 引出DP滚动数组思想，每次只依赖前两个状态
3. **"如果房屋金额有负数呢？"** — 引出初始化和边界条件的变化
4. **"能否用记忆化搜索（自顶向下）实现？和自底向上DP有什么区别？"** — 测试递归+备忘录 vs 迭代DP的理解
5. **"这个状态转移方程的本质是什么类型的DP？"** — 答案是一维线性DP/序列DP
