# 【阶跃星辰面经】蒙特卡洛模拟与时序差分估计，核心差异在哪里

## 一、两种方法对比

| 维度 | 蒙特卡洛（MC） | 时序差分（TD） |
|------|--------------|--------------|
| 更新时机 | **轨迹结束后** | **每一步** |
| 更新目标 | 实际总回报 G_t = Σγ^k r_{t+k} | r_t + γV(s_{t+1})（TD 目标） |
| 偏差 | **无偏**（用真实回报） | **有偏**（用 V 估计） |
| 方差 | **大**（单条轨迹噪声） | **小**（只看一步） |
| 是否 bootstrap | 否（不依赖自身估计） | **是**（用 V(s') 估计） |
| 连续任务 | 不适用（无终止） | 适用 |
| 样本效率 | 低（等整条轨迹） | 高（每步可学） |

## 二、核心差异详解

### 差异1：更新时机——"事后"vs"边走"

```
MC：等轨迹 s0→a0→r0→s1→a1→r1→...→sT 全部结束
    用实际总回报 G_t = r_t + γr_{t+1} + ... + γ^{T-t}r_T 更新 V(s_t)

TD：每走一步 s_t→a_t→r_t→s_{t+1}
    立即用 r_t + γV(s_{t+1}) 更新 V(s_t)
    （不等后续，用已有的 V(s_{t+1}) 估计）
```

### 差异2：偏差——无偏 vs 有偏（bootstrap）

```
MC：G_t 是实际发生的回报 → 无偏估计
TD：r_t + γV(s_{t+1}) 用了"V(s_{t+1})这个估计值"
    → 如果 V 不准，TD 目标也不准 → 有偏
    → 这种"用自身估计更新自身"叫 bootstrap
```

### 差异3：方差——大 vs 小

```
MC：G_t = Σ γ^k r_{t+k} 累积了 T-t 步奖励
    → 每步奖励都有随机性 → 累积方差大
    → 不同轨迹的 G_t 差异大

TD：r_t + γV(s_{t+1}) 只含 1 步随机奖励 r_t
    → V(s_{t+1}) 是平滑估计（已平均多条轨迹）
    → 方差小，更新稳定
```

## 三、用开车举例

```
从 A 开车到 B，要估"从 A 出发的预期时间 V(A)"

MC 方法：开完整程（A→B），看实际花了多久 → 更新 V(A)
  - 无偏：实际时间就是真实值
  - 方差大：堵车/红绿灯让每次实际时间差异大
  - 必须到达 B 才能更新

TD 方法：开到下一站 C，看 A→C 用了多久 + 从 C 到 B 的预估
  - V(A) ← 实际(A→C) + V(C)
  - 有偏：V(C) 是估计值，可能不准
  - 方差小：只看 A→C 这一段
  - 到 C 就能更新，不用等到 B
```

## 四、为什么 TD 更常用（RLHF/PPO 用 TD）

1. **样本效率高**：每步都能学，不用等轨迹结束
2. **方差小**：训练稳定（方差大→梯度震荡）
3. **适用连续任务**：不要求轨迹终止
4. **代价是偏差**：用 GAE（λ 插值）缓解

**RLHF 实践**：PPO 用 GAE（TD 的多步融合版），不用纯 MC。

## 五、n-step TD 和 TD(λ)：折中

```
1-step TD: V(s_t) ← r_t + γV(s_{t+1})                    （高偏差低方差）
2-step TD: V(s_t) ← r_t + γr_{t+1} + γ²V(s_{t+2})
...
n-step TD: V(s_t) ← Σ_{k=0}^{n-1} γ^k r_{t+k} + γ^n V(s_{t+n})
∞-step (MC): V(s_t) ← Σ γ^k r_{t+k}                       （低偏差高方差）

TD(λ) / GAE：用 (γλ)^k 衰减权重融合所有 n 步 → 最优折中
```

## 六、加分点

- 说出 **bootstrap 是 TD 的关键特征**：用自身估计更新自身（MC 不 bootstrap）
- 说出 **MC 无偏但方差大，TD 有偏但方差小**——偏差方差权衡
- 说出 **TD 收敛到 Bellman 方程的解**（在表格法下可证明）

## 七、雷区

- ❌ "MC 和 TD 只是更新频率不同" → 核心是 bootstrap 和偏差方差
- ❌ "TD 一定比 MC 好" → TD 有偏，某些场景 MC 更稳
- ❌ 混淆 TD target 和 TD error → target 是 r+γV(s')，error 是 target - V(s)

## 八、扩展

- **TD(0)**：1 步 TD，最简单的 TD
- **Q-learning**：off-policy 的 TD 控制，用 max Q(s',a') 做 bootstrap
- **SARSA**：on-policy 的 TD 控制，用实际 Q(s',a') 做 bootstrap
- **为什么 RLHF 用 GAE 而非纯 MC**：MC 方差大导致 PPO 训练不稳，GAE 平衡
