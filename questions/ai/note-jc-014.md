# 【阶跃星辰面经】动态规划核心思想 + 最短路逆推子问题

## 一、动态规划核心思想

**核心**：把大问题拆成子问题，**存储子问题的解**，避免重复计算。

**适用条件**（必须同时满足）：
1. **最优子结构**：大问题的最优解包含子问题的最优解
2. **重叠子问题**：子问题被反复计算（暴力递归会指数爆炸）

## 二、DP 的两大要素

```
[1] 状态定义：dp[...] 代表什么
    例：dp[i] = 以第 i 个元素结尾的最长子序列长度
    
[2] 状态转移方程：dp[i] 怎么从子问题算出来
    例：dp[i] = max(dp[j] + 1) for all j < i 且 arr[j] < arr[i]
```

## 三、经典例子：斐波那契（重叠子问题）

```python
# 暴力递归：O(2^n)（重复计算 fib(3) 无数次）
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)

# DP（记忆化）：O(n)（每个 fib(i) 只算一次）
memo = {}
def fib(n):
    if n <= 1: return n
    if n in memo: return memo[n]
    memo[n] = fib(n-1) + fib(n-2)
    return memo[n]

# DP（迭代）：O(n)
def fib(n):
    if n <= 1: return n
    dp = [0]*(n+1)
    dp[1] = 1
    for i in range(2, n+1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

**为什么 DP 快**：暴力递归 fib(5) 会算 fib(3) 两次、fib(2) 三次...指数级重复。DP 存起来只用算一次。

## 四、经典例子：0-1 背包（最优子结构）

```
问题：n 个物品（重量w[i]、价值v[i]），背包容量 W，求最大价值。

状态定义：dp[i][j] = 前 i 个物品、容量 j 时的最大价值
状态转移：
  不选第 i 个：dp[i][j] = dp[i-1][j]
  选第 i 个：  dp[i][j] = dp[i-1][j-w[i]] + v[i]  （j >= w[i]）
  取最大：dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])

最优子结构：dp[i][j] 的最优来自 dp[i-1][...] 的最优。
```

## 五、最短路问题的逆推解法

### 问题定义
给定有向图，求从起点 s 到终点 t 的最短路径。

### 逆推解法（从终点倒推）

**子问题定义**：
```
dp[i] = 从节点 i 到终点 t 的最短距离

边界：dp[t] = 0（终点到终点距离 0）
目标：dp[s]（起点到终点最短距离）

状态转移（逆推）：
  dp[i] = min over all edges (i→j):  weight(i,j) + dp[j]
  
  即：从 i 出发，看所有相邻节点 j，
      "走 i→j 这条边 + 从 j 到终点的最短距离"，
      取最小。
```

### 示例

```
图：
  s --3--> A --2--> t
  s --5--> B --1--> t
  A --4--> B

逆推：
  dp[t] = 0
  dp[B] = w(B,t) + dp[t] = 1 + 0 = 1
  dp[A] = min(w(A,t)+dp[t], w(A,B)+dp[B])
        = min(2+0, 4+1) = min(2, 5) = 2
  dp[s] = min(w(s,A)+dp[A], w(s,B)+dp[B])
        = min(3+2, 5+1) = min(5, 6) = 5

最短路径：s→A→t，距离 5
```

### 子问题是什么

```
子问题 = "从某个节点到终点的最短距离"
  dp[A] 的子问题是 dp[t] 和 dp[B]（A 的邻居）
  dp[s] 的子问题是 dp[A] 和 dp[B]（s 的邻居）

最优子结构：dp[s] 的最优 = min(边(s,邻居) + 邻居的最优 dp[邻居])
重叠子问题：如果有多条路径经过同一节点，该节点的 dp 值被多次复用
```

## 六、逆推 vs 顺推（Dijkstra）

```
逆推（DP）：从终点倒推，适合 DAG（无环图）
  dp[i] = min(w(i,j) + dp[j])
  
顺推（Dijkstra）：从起点正向扩展，适合带非负权的任意图
  用优先队列每次取最近节点扩展

DAG 最短路用逆推 DP（拓扑序处理）更简单；
有环图用 Dijkstra（DP 会循环依赖）。
```

## 七、加分点

- 说出 **DP 的两个适用条件**：最优子结构 + 重叠子问题
- 说出 **状态定义和状态转移方程是 DP 的核心**
- 说出 **最短路逆推的子问题是"从某节点到终点的最短距离"**

## 八、雷区

- ❌ 混淆贪心和 DP：贪心不回退（局部最优），DP 考虑所有子问题（全局最优）
- ❌ 状态定义错误：dp[i] 含义不清导致转移方程写错
- ❌ 忽略边界条件：dp[0] / dp[t] 等初始值要正确

## 九、扩展

- **自顶向下（记忆化递归）vs 自底向上（迭代）**：两者等价，迭代通常更快（无递归开销），记忆化更直观
- **空间优化**：背包问题 dp[i][j] 只依赖 dp[i-1][...]，可压缩成一维 dp[j]
- **区间 DP / 树形 DP / 状压 DP**：按状态结构分类的高级 DP 技巧
